# 共通ライティングスタイル

## 基本原則

- **事実と解釈を分ける**: 観察した事実と、そこから導く解釈は明示的に区別する
- **箇条書き優先**: 長文の段落より、箇条書きで要点を整理する
- **結論を先に**: 結論・判断を冒頭に置き、根拠を後に続ける
- **敬語レベルを統一**: 社内向けは丁寧語、役員向けはより簡潔・格式を高める

## 避けること

- 曖昧な表現（「頑張っていた」「なんとなく」）
- 主語が不明な文（「検討された」→ 誰が？）
- 感情的な評価語（「問題がある」→ 具体的にどの行動が？）

## フォーマット

```
【結論】
〇〇を推奨する。

【根拠】
- 事実1: ...
- 事実2: ...

【補足・懸念点】
- ...
```
