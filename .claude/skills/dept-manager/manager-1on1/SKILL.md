# manager-1on1

## 役割

1on1 面談に関する以下の 3 つを担当する。

1. **準備**: アジェンダ・事前確認事項・問いの候補を作成する
2. **実施支援**: 面談中に使えるチェックポイントを提示する
3. **フォローアップ**: 面談後の要点整理・アクション一覧を作成する

## ステップ 0：対象者の確認とデータ読み込み

Skill 起動時、対象者が指定されていない場合は **必ず最初に確認する**。

```
誰との1on1ですか？（例：Aさん、Bさん）
```

対象者が決まったら `team_data/{名前}/` に以下のファイルが存在するか確認する。

| ファイル | 内容 |
|---|---|
| `monthly-schedule.md` | 今月の予定・タスク |
| `career-goals.md` | キャリア目標 |
| `term-job-plan.md` | 今期の職務予定 |

### データが存在しない場合

以下のメッセージを出してスキルを**一時停止**する。

```
{名前}さんの team_data が見つかりませんでした。

1on1 を実施するには、先にデータを用意してください。

【必要なファイル】
  team_data/{名前}/monthly-schedule.md  ← 今月の予定・タスク
  team_data/{名前}/career-goals.md      ← キャリア目標
  team_data/{名前}/term-job-plan.md     ← 今期の職務予定

ファイルを作成したら、もう一度「1on1の準備をして」と伝えてください。
```

### データが存在する場合

3ファイルを読み込み、内容を踏まえて以降のステップを実行する。

## ステップ 1：準備（面談前）

読み込んだデータを踏まえ、以下を出力する。

```
【対象者】{名前}さん

【面談目的】
...（キャリア目標・今期業務から自動導出）

【アジェンダ（30分想定）】
0:00-0:05  近況・体調確認
0:05-0:15  今月のタスク進捗確認（monthly-schedule より重点タスクを列挙）
0:15-0:25  課題・懸念の深掘り
0:25-0:30  ネクストアクション確認

【今月の重点タスク（確認ポイント）】
- ...（monthly-schedule.md の優先度：高 タスクを引用）

【キャリアとの接続（問いの素材）】
- ...（career-goals.md と今期業務のギャップや進展を抽出）

【問いの候補】
- ...（オープン質問：相手の考えを引き出す）
- ...（具体化の質問：状況をより明確にする）
- ...（キャリア視点の質問：成長実感・課題感を掘り下げる）
```

## ステップ 2：フォローアップ（面談後）

面談内容を受け取ったら以下を出力し、`outputs/one-on-ones/YYYY-MM-DD_{名前}.md` に保存する。

```
【面談要点】
- ...

【合意・確認事項】
- ...

【ネクストアクション】
| アクション | 担当 | 期限 |
|---|---|---|
| ... | ... | ... |

【次回への申し送り】
- 継続して確認すること:
- キャリア観点で追うこと:
```
