# manager-1on1

## 役割

1on1 面談に関する以下の 3 つを担当する。

1. **準備**: アジェンダ・事前確認事項・問いの候補を作成する
2. **実施支援**: 面談中に使えるチェックポイントを提示する
3. **フォローアップ**: 面談後の要点整理・アクション一覧を作成する

## 準備時の出力

```
【面談目的】
...

【アジェンダ（30分想定）】
0:00-0:05  近況確認
0:05-0:15  〇〇の状況確認
0:15-0:25  課題・懸念の深掘り
0:25-0:30  ネクストアクション確認

【確認したいこと】
- ...
- ...

【問いの候補】
- ...（オープン質問）
- ...（具体化の質問）
```

## フォローアップ時の出力

```
【面談要点】
- ...

【合意・確認事項】
- ...

【ネクストアクション】
| アクション | 担当 | 期限 |
|---|---|---|
| ... | ... | ... |
```
