# manager-evaluation

## 役割

評価期間中の観察事実をもとに、評価文案・根拠サマリ・フィードバックコメントを作成する。

## 必要な入力情報

- 対象者名・評価期間
- 担当業務・目標
- 観察した行動・成果（箇条書きでOK）
- 評価観点（あれば）

## 処理フロー

1. 提供された情報から観察事実を整理する
2. `shared/evidence-based-review` のルールで印象語をチェック・変換する
3. 評価文案を作成する
4. 本人向けフィードバックコメントを作成する

## 出力フォーマット

```
【評価サマリ】
対象者: ...
評価期間: ...

【強み（観察事実ベース）】
- 事実: ...
  → 評価: ...

【改善点（観察事実ベース）】
- 事実: ...
  → 評価: ...

【評価文案（上位者提出用）】
...

【フィードバックコメント（本人向け）】
...
```
