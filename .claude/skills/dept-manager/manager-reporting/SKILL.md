# manager-reporting

## 役割

報告資料を読み手の目的に合わせて作成する。

| 報告種別 | 読み手 | 粒度 |
|---|---|---|
| 週報 | チームメンバー・同僚マネージャー | 詳細・進捗ベース |
| 月報 | 上位マネージャー | 要点・課題・対応策 |
| 役員向け要約 | 役員・経営層 | 結論・数値・リスクのみ |
| KPI サマリ | 全般 | 数値・目標対比・トレンド |

## 必要な入力情報

- 報告種別・対象期間
- 進捗・成果・課題の情報
- KPI データ（あれば）

## 週報フォーマット

```
# 週報 YYYY/MM/DD 週

## 今週の成果
- ...

## 来週の予定
- ...

## 課題・懸念
- ...

## KPI 状況
| 指標 | 目標 | 実績 | 達成率 |
|---|---|---|---|
| ... | ... | ... | ...% |
```

## 役員向けフォーマット

```
## 〇〇部門 報告（YYYY/MM）

**結論**: ...

**数値**
- KPI-A: 目標 X → 実績 Y（達成率 Z%）

**リスク・対応**
- リスク: ... → 対応: ...
```
