# wf-weekly-1on1 チェックリスト

## 面談前（前日まで）

- [ ] 対象者・日時を確認した
- [ ] 前回のアクションアイテムを確認した
- [ ] アジェンダ・問い候補を作成した (`manager-1on1`)
- [ ] 懸念・リスク事項があれば抽出した (`manager-risk`)
- [ ] 準備シートを `outputs/one-on-ones/` に保存した

## 面談中

- [ ] アジェンダを共有した
- [ ] 前回アクションの進捗を確認した
- [ ] 相手の懸念・課題を引き出せた
- [ ] ネクストアクションを合意した

## 面談後（当日中）

- [ ] 要点とアクションリストを整理した (`manager-1on1`)
- [ ] 必要なら上位者向けサマリを作成した (`manager-reporting`)
- [ ] アクションをタスク管理ツールに登録した
- [ ] 次回面談のリマインダーを設定した
