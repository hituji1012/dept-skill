# wf-weekly-1on1

## 概要

毎週の 1on1 を効率よく進めるための Workflow。
準備 → 面談 → フォローアップ の 3 フェーズをカバーする。

## フロー

```
[起動時]
wf-weekly-1on1        対象者を確認する（未指定なら質問）
       ↓
wf-weekly-1on1        team_data/{名前}/ の存在を確認する
       ↓（データなし）→ データ作成を案内して一時停止
       ↓（データあり）
[前日まで]
manager-1on1          team_data を読み込み、アジェンダ・問い候補・準備シートを作成
       ↓
manager-risk          (任意) 懸念事項・リスクの事前抽出

[面談後]
manager-1on1          要点整理・アクションリストを作成
       ↓
manager-reporting     (任意) 上位者向けに 1〜2 行で要点を圧縮
```

## 起動時の対象者確認

対象者が指定されていない場合、以下のように確認する。

```
誰との1on1ですか？（例：Aさん、Bさん）
```

対象者が決まったら `team_data/{名前}/` を確認する。

- **ファイルあり** → そのまま準備フェーズへ進む
- **ファイルなし** → 以下を案内して停止する

```
{名前}さんの team_data が見つかりませんでした。

以下の3ファイルを用意してから再度実行してください。

  team_data/{名前}/monthly-schedule.md  ← 今月の予定・タスク
  team_data/{名前}/career-goals.md      ← キャリア目標
  team_data/{名前}/term-job-plan.md     ← 今期の職務予定

ファイルの内容についてサポートが必要な場合はお知らせください。
```

## 使い方

```
# 対象者を指定して起動
/wf-weekly-1on1 Aさん

# 対象者を指定せずに起動（質問される）
/wf-weekly-1on1
```
