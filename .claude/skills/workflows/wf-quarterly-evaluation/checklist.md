# wf-quarterly-evaluation チェックリスト

## 評価前（証拠収集）

- [ ] 評価期間・評価観点を確認した
- [ ] 対象者の目標・KPI を確認した
- [ ] 観察した行動・成果を箇条書きで収集した
- [ ] 1on1 メモ・プロジェクト記録など証拠を参照した

## 評価ドラフト作成

- [ ] `manager-evaluation` で評価ドラフトを作成した
- [ ] `shared/evidence-based-review` のルールでチェックした
- [ ] 印象語を事実ベースの記述に変換した
- [ ] ポジティブ・ネガティブ双方の証拠がある

## 判断整理

- [ ] 昇格・育成・配置などの判断オプションを整理した
- [ ] 各オプションの根拠・リスクを整理した

## 上位者向け資料

- [ ] `manager-reporting` で上位者向けサマリを作成した
- [ ] 結論・根拠・リスクが 1 ページに収まる形になっている
- [ ] `outputs/evaluations/` に保存した
