# Aさん 今期職務予定（2026年 Q1：1月〜3月）

## 担当業務概要

クラウドインフラ基盤の設計・構築・運用を中心に、チームの技術力底上げとガバナンス整備を担う。

## 主要業務と目標

### 1. AWS移行プロジェクト Phase2（技術リード）

- **概要**：オンプレミスサーバー群をAWSへ移行する第2フェーズ
- **役割**：アーキテクチャ設計・実装・レビュー
- **今期ゴール**：設計書確定 → 実装完了 → 動作確認まで
- **進捗目安**：3月末までに移行完了

### 2. クラウドガバナンス方針策定

- **概要**：部門横断でのクラウド利用ルール・コスト管理ポリシーを整備
- **役割**：ドラフト作成・関係部署との調整（★苦手領域の実践）
- **今期ゴール**：ドラフト完成 → 1部署との合意取得
- **備考**：マネージャーサポートのもとでファシリテーション練習

### 3. 後輩メンバー技術支援

- **概要**：チーム内メンバー2名（C, D）の技術相談対応・ペアワーク
- **今期ゴール**：各自が1タスクを独立完遂できるレベルへ引き上げ

### 4. IAMセキュリティ監査・整備

- **概要**：既存IAMポリシー・ロールの棚卸しと最小権限原則への整合
- **今期ゴール**：棚卸し完了・リスク高箇所の修正
